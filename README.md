# FUTURE_CS_01-FIT-APR26-CS7994
# Website Vulnerability Assessment Report

## Project Overview

This project demonstrates a professional Vulnerability Assessment conducted on a public-facing website using passive and ethical security testing techniques.

The assessment focused on identifying common web security weaknesses including:

- Missing security headers
- Information disclosure
- Weak cookie configuration
- Exposed services
- Outdated technologies
- SSL/TLS misconfigurations

All testing activities were conducted within a read-only scope without exploitation or harmful actions.

---

# Objective

The objective of this assessment is to:

- Analyze a public website for security weaknesses
- Identify potential business risks
- Explain vulnerabilities in simple language
- Recommend practical remediation steps
- Demonstrate professional security auditing skills

---

# Scope

## Allowed Activities
- Passive scanning
- Header analysis
- Public page inspection
- SSL/TLS inspection
- Cookie analysis

## Restricted Activities
- Exploitation
- Login bypass
